World Wide Web's skapelse

Word Wide Web (www) skapades 1989 av Tim Bernes-Lee så att man skulle kunna surfa på webben. Han skapar en webbplatts och webbserver som gör det möjligt att söka och använda olika hemsidor.

allmänheten får åtkomt till webben

1993 så blir de möjligt att använda sig av hemsidor och länkar och det är nu som webbsidor verkligen börjar sprida sig. Det blir också väldigt populärt att göra egna personliga hemsidor men det var även företag som använde sig av webbplattser. Man kan nu ladda ner document, bilder och program via HTML länkar.
